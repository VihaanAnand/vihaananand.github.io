# Parametric — Colour Reference

| Variable       | Value              | Name                  | Used for                                                                 |
|----------------|--------------------|-----------------------|--------------------------------------------------------------------------|
| `--ink`        | `#1c1c1e`          | Near black            | Primary text, headings, labels, active button states                     |
| `--accent`     | `#0091ff`          | Bright blue           | Buttons, links, section labels, icons, focus rings, testimonial border   |
| `--accent-lt`  | `#ddf0ff`          | Pale blue             | Eyebrow tag background, step numbers (decorative large numerals)         |
| `--surface`    | `#f2f2f7`          | Off white / light grey| Alternating section backgrounds, testimonial card backgrounds            |
| `--white`      | `#ffffff`          | White                 | Page background, subject cards, form inputs                              |
| `--border`     | `#e2e8f0`          | Light grey            | Card borders, form input borders, divider lines                          |
| —              | `#94a3b8`          | Cool grey             | Input placeholder text, dropdown chevron icon (hardcoded, not a var)     |
| —              | `rgba(0,145,255,.12)` | Blue (12%)         | Focus ring glow on form inputs                                           |
| —              | `rgba(0,0,0,.07)`  | Black (7%)            | Subject card hover shadow                                                |
| —              | `rgba(0,145,255,.35)` | Blue (35%)         | Primary button hover shadow                                              |
